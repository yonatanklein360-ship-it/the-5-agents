# לוג חיפושים — חן

קובץ זה מתעד את כל החיפושים שחן ביצעה ברשת. **לפני כל חיפוש** חן בודקת כאן (באמצעות `Grep` על מילות המפתח) אם כבר חיפשה משהו דומה ב-30 הימים האחרונים, כדי למנוע עבודה כפולה. **אחרי כל חיפוש** היא מוסיפה entry חדש בפורמט שלמטה.

## פורמט entry

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
